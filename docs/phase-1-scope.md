# Salimon 1차 개발 범위

## 이번 구현

- pnpm 모노레포 구성
- Next.js 웹앱 기반 로컬 MVP
- MobX 상태 관리
- 공통 타입, 카드 문자 파싱, 마스킹, 날짜/금액 도메인 로직
- 로컬 저장소 기반 거래 CRUD, 월간 캘린더, 카테고리 관리
- 공동 가계부 초대 코드 시뮬레이션
- 카드 문자 후보함, 기타 등록, 제외, 나중에 처리
- 카드 문자 예시 제출 및 마스킹 미리보기
- Supabase DB/RLS 마이그레이션 초안
- Android 네이티브 구현을 위한 모바일 패키지 스텁

## 1차 이후 연결

- Supabase 프로젝트 연결 및 마이그레이션 적용

- Android 암호화 로컬 저장소 연결
